DOMAIN SUMMARIES



Domain 1

Domain 1 is the shortest out of them all. In this domain we learn to create an on screen popup depending on the Javascript. If we write script we can create a popup that says "This website supports Javascript", if we write noscript we can create another popup that says "This website does not support Javascript


Domain 2

In Domain 2, we learn the basics of CSS (cascading style sheets). CSS is used to decorate a website, decorating a website can be very important to catching the reviewers attention. CSS can be used to change fonts, colors, add shapes, borders and organize your website the way you want.


Domain 3

In Domain 3, we learn the basics of html, there are many tags that should be known if you want to create a good website, we learn how to make tables, lists, add images and links. We also learn about divs which can better organize your website by breaking it down into parts so it is easier to read and navigate. Navigation can also be achieved through bookmarks which takes you to a specfic part of a page if you have a big one. Lastly we learn to make a contact us page where users can put in their information where we can transfer to a different part of the page.


Domain 4

In Domain 4, we learn ways to embed files onto your webpage, the files we embeded consist of images, mp3s, and mp4s. With these things we learn to add links connected to them. We also put on controls to skip, pause or resume a video.


Domain 5

In Domain 5, we learn to organize and better decorate our web page. Using CSS, we change fonts, colors, and organize our page. In css we learn to move stuff using relative and absolute position. We change the visibility so it can make something appear or disappear. Using fonts, letter spacing, and borders we are better able to design a webpage to become a lot more personalized and interesting


Domain 6

In Domain 6, we learn to put everything we learned together in a way. Using all types of CSS and HTML to decorate a webpage. With the use of comments, we are able to temporarily turn off code without deleting it which can come handy to see if something is working as it should.
